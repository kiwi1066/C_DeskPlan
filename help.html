Desk Planner — User Guide

Overview

The Desk Planner lets you visually assign desks to teams across the working week using an interactive floor plan. Import teams, allocate desks by day, and export your plan as a CSV or image for sharing and archiving.

Basic Workflow

  1. Select a Day from the day dropdown.
  2. Add teams — either via Import Teams (CSV) or the Add Team button.
  3. Click one or more desks on the floor plan to select them. Use Multi mode or hold Ctrl to select multiple desks at once.
  4. Click a team chip in the Teams legend — the selected desks are assigned to that team.
  5. Repeat for each day. Use Copy Day to duplicate Monday's plan across the week.
Tip: Desk colours match the team legend. Hover over a coloured desk to see the team name as a tooltip.

Toolbar Reference

File

👥 Import Teams Imports a two-column CSV (id,name, e.g. T1,Finance). Rebuilds the team legend automatically. Replaces any existing teams.
📂 Import Imports a previously exported desk-plan CSV, restoring both desk assignments and team names across all days.
⬇ Export Exports the current desk plan to CSV (desk, mon…fri, plus team names). You will be prompted for a file name.
📸 Export Image Saves the current floor plan — including team colours and an allocation summary — as a PNG image.

Edit

📋 Copy Day Copies all desk assignments from the current day to one or more other days (selected via checkboxes in the pop-up).
↩ Undo Reverts the last desk-assignment change. Supports up to 50 undo steps.
🧽 Clear Desks If desks are selected, clears only those desks for the current day. If nothing is selected, clears all desks for the current day.
🧹 Clear All Clears all desk assignments and removes all teams, without reloading the page. Saved state is also cleared.
🗑 Reset Completely resets the application and reloads the page. All data is lost.

Select & Teams

🧩 Multi Toggles multi-select mode. When ON (green), clicking desks adds them to the selection without needing to hold Ctrl.
➕ Add Team Adds a new team manually. You will be prompted for a name. The system assigns the next available ID (T1, T2 … Tn).
❓ Help Opens this guide in a new tab.

Teams

  • Import teams from a CSV file with columns id and name.
  • Or add teams one at a time with the Add Team button.
  • Click a team chip in the legend to assign selected desks to that team.
  • Each team is automatically given a unique colour.
CSV format for teams:
id,name
T1,Finance
T2,Engineering

Desk Allocation Summary

The summary panel (below the Teams legend) shows:

  • Total desks on the floor
  • How many are assigned vs free for the selected day
  • A per-team desk count

Compare Mode

Switch the Mode dropdown to Compare to view two days side-by-side. Editing is disabled in this mode — switch back to Single to make changes.

Tips & Best Practices

  • Export your plan regularly — it saves your work as a portable CSV.
  • Use consistent team IDs (T1, T2 …) if you manage multiple floors.
  • Use Copy Day to quickly roll Monday's plan forward for the week, then tweak individual days.
  • The exported CSV can be re-imported at any time to restore a previous state.
